Depends : jdk,yuicompressor

get yuicompressor

wget -c wget https://github.com/yui/yuicompressor/releases/download/v2.4.8/yuicompressor-2.4.8.jar

How to compress?

java -jar yuicompressor-2.4.8.jar --type css style.css > style.css
