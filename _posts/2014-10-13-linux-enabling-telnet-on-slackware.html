1. Check package telnet, with slack-utils.

sll telnet

2. On Package inetd, uncheck # telnet

sudo vi  /etc/inetd.conf

telnet    stream  tcp     nowait  root    /usr/sbin/tcpd    in.telnetd

3. Restart inetd

sudo inetd -d

4. Testing localhost telnet

telnet 127.0.0.1
