membuat listener #oracle di #igosnusantara

disable    host :
xhost +

masuk ke user oracle :
su oracle

lalu membuat listener  :
netca

Oracle Net Configuration Assistant
[1] listener configuration, next
[2] add, next
[3] berikan nama untuk listener, disini    menggunakan LISTENER
[4] pilih protokol tcp,    next
[5] tentukan port, gunakan port    standar    1521
[6] pilih opsi no, next
[7] next
[8] finish

mengecek apakah    listener sudah bisa dijalankan :
lsnrctl start
