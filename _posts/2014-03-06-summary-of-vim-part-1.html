Mode perintah : Esc

Insert : Esc + i.

Rumus Umum :

Esc + Nomer + Perintah + Etc

Move Cursor with One Space:

Esc + h (ke kiri), l (ke kanan), j (ke bawah),k (ke atas).

Move Cursor on word :

Esc + w (ke kata berikut dengan posisi kursor pada awal kata).

Esc + e (ke kata berikutnya dengan posisi kursor pada akhir kata).

Esc + b (ke kata sebelumnya dengan posisi kursor pada awal kata).

Move Cursor a Few Words :

Esc + 3 + w

(Pindah kursor melewati 3 huruf, dan posisi kursor pada awal kata setelah melewati 3 huruf)

Esc + 2 + e (Pindah kursor melewati dua huruf, dan posisi kursor pada akhir kata setelah melewati 2 huruf)

Esc + 2 + b

(Pindah kursor melewati 2 huruf dibelakang dengan posisi kursor pada awal kata)

Move Cursor a Few Characters :

Esc + 2 +  l (Pindah Kursor melewati 2 karakter ke sebelah kiri).

Note : Dapat dikombinasikan dengan arrow atau h,j,k,l.
