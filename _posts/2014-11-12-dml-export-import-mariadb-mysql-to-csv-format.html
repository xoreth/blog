DML Export Import Mariadb (Mysql) to csv Format

Export Content Table to CSV

mysql -u root -p nama_db -e “select  column_1, column_2, column_3 from Tabel_target ” | sed ’s/t/“,”/g;s/^/“/;s/$/”/’ > text.csv

Export Database to CSV

mysqldump -u root -p nama_db > export.db

regex :

sed ’s/t/“,”/g;s/^/“/;s/$/”/’ (regex CSV Format)
