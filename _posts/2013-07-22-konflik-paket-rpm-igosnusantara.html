Misal nama file yang mendapatkan konflik adalah filesystem, file binary executable elf pada /usr/bin/ adalah sama dengan file yg terdapat pada paket net-tools. untuk melihat daftar file pada nama paketÂ  :

rpm -ql namaPaket | less

dan untuk melihat file-file rpm yang akan diinstallkan (konflik)

rpm -qpl /direktoridimanapaketberada/namapaket | less

Untuk install agar file yang konflik dapat ditangani maka :
rpm --force -i /direktoridimanapaketberada/namapaket

  • #warning : bersifat replace, menindih, file lama akan terhapus.
