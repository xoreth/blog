Instalasi :

$ git clone git://github.com/soimort/google-translate-cli.git
$ cd google-translate-cli/
$ make install

 

Masukan ke /usr/bin/

$sudo cp -R translate* /usr/bin

 

Penggunaan :

#Inggris-Indonesia

translate {en=id} "I hate You"

#Indonesia-Inggris

translate {id=end} "saya telah lapar"
