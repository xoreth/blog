1. Listing image

VBoxManage list vms

output

"CentOS6.7" {59051034-c1c8-44cb-b985-cd641b790690}


2. Create package

vagrant package --base 59051034-c1c8-44cb-b985-cd641b790690 --output centos.box

3. add to vagrant

vagrant box add centos.box --name centos

4. Check vagrant box

vagrant box list
