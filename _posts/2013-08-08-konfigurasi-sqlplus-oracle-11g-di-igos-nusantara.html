Konfigurasi #sqlplus #oracle 11g di #igosnusantara

tools client oracle 11g sqlplus tidak bisa diarahkan keatas dan tidak mempunyai tab completion.
Solusi :

sudo yum install rlwrap

setalah    itu masuk ke user oracle,
#noted
http://blog.wajatmaka.com/post/57150322864/oracleindex-1-instalasi-konfigurasi-servis-oracle-11g

lalu membuat KAMUS untuk tab completion    di /home/oracle/
nano ~/.library1.dat

SELECT FROM ORDER WHERE JOIN TABLE VIEW PACKAGE PROCEDURE FUNCTION IDENTIFIED BY GROUP ALTER DELETE INSERT UPDATE
SHOW DESCRIBE ON ALL IN  INTO

lalu buat history sqlplus
touch ~/.sqlplus_history   

Setelah membuat    kamus dan history, buat env alias untuk sqlplus di /home/oracle/.zshrc atau /home/oracle/.bashrc

alias sqlplus=‘rlwrap -i -f ~/.library1.dat sqlplus’

lalu masuk ke sqlplus. temporari perintah dapat    diarahkan dengan keyboard arah atas,
dan dapat melakukan tab    completion dengan menekan tab dua kali.
