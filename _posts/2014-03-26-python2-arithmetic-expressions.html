Numbers, Operator dan Expression

print 1, 2.1324, -1, -3

Output

1 2.1324 -1 -3
Melihat tipe data yang digunakan dengan cara

print type(1), type(2.1324)

Output

<type ‘int’> <type 'float’>

Deklarasi tipe data bersifat dinamis.

print type(2.1323)

Output

<type 'float’>

Cara konversi tipe data float ke integer

print int(2.1323)

Output

2

Sebaliknya dari int ke float

print float(3)

Output

3.0

batas panjang length pada python 2.7.5 dibatasi 11 angka dibelakang koma atau hanya dibatasi 12 angka secara visual.

print 3.1415926535897932384626433832795028841971

Arithmetic Operator

+ = tambah

- = kurang

/ = bagi

* = kali

** = pangkat

print 1+1,3-1,15/5, 2*3, 2**5

Output

2 2 3 6 32
