Menjalankan service database #oracle di #igosnusantara

disable    host :
xhost +

masuk ke user oracle :
su oracle

pada /etc/oratab ubah
basedata:/opt/oracle/product/11.2.0/db_1:N
menjadi
basedata:/opt/oracle/product/11.2.0/db_1:Y

lalu pada  /opt/oracle/product/11.2.0/db_1/bin/dbshut
cari dan ganti,
ORACLE_HOME_LISTNER=$1
menjadi
ORACLE_HOME_LISTNER=$ORACLE_HOME

dan pada /opt/oracle/product/11.2.0/db_1/bin/dbstart
cari dan ganti,
ORACLE_HOME_LISTNER=$1
menjadi
ORACLE_HOME_LISTNER=$ORACLE_HOME

setelah    konfigursi diatas selesai, jalankan service
[1] jalankan LISTENER
lsnrctl start
[2] jalankan service DATABASE
dbstart

Untuk mengecek, masuk sebagai sys pada oracle :
sqlplus  
lalu pada username isi dengan :
/as sysdba

Untuk mematikan service :

[1]matikan service database

dbshut

[2]matikan listener

lsnrctl stop
