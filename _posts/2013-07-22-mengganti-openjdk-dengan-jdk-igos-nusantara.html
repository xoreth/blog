Jika mempunyai OpenJDK dan JDK maka harus memakai salah satu dari meraka. cara memilihnya dapat menggunakan alternatives :

alternatives –install /opt/jdk java  /opt/jdk/bin/java 90
alternatives –install /opt/jdk javac /opt/jdk/bin/javac 90

alternatives –config java

lalu pilih jdk

alternatives –config javac

lalu pilih yang dimiliki jdk

Selesai, lalu cek :

➜  ~  java -version
java version “1.7.0_25”
Java™ SE Runtime Environment (build 1.7.0_25-b15)
Java HotSpot™ 64-Bit Server VM (build 23.25-b01, mixed mode)
