[Fixing] Evince cannot display icon

Evince (PDF viewer from gnome).

Masalah : Evince tidak dapat menampilkan icon.

Solusi :

buat file ~/.config/gtk-3.0/settings.ini

[Settings]
gtk-key-theme-name = Emacs
gtk-fallback-icon-theme = gnome
gtk-theme-name = oxygen-gtk
