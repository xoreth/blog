Membutuhkan easy_install (https://pypi.python.org/pypi/setuptools#downloads) sebagai software installer module.

Untuk memudahkan manajemen baik instalasi, konfigurasi dan menghapus modul/paket yang akan dipasang maka pasang python package management system. => pip

Package Management pip

Pada pip, bisa melihat module yang dipakai, mencari module yang akan dipasang dan lain-lain.

pip —help

Adapun melihat package/module yang terinstall selain menggunakan pip pada python. masuk pada mode python.

»help(“modules”)

#Noted

»help(“”) adalah salah satu manual dokumentasi python.

»dir(“”) mengembalikan fungsi daftar semua nama dalam direktori obyek apa pun yang Anda berikan sebagai argumen.
