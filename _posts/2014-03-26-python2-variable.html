# Variabel - penampung nilai
# Digunakan untuk menghindari nilai-nilai recomputing dan memberikan nilai pada nama yang membantu pembaca memahami kode

# Valid nama variabel - terdiri dari huruf, angka, garis bawah (_)
# Dimulai dengan huruf atau underscore ( _ )
# Case sensitive

# Valid - Ninja, Ninja, n_i_n_j_a
# Invalid - 1337, 1337ninja

# Nama variabel yang digabungkan dengan underscore _
# Valid - elite_ninja, leet_ninja, ninja_1337
# Invalid 1337_ninja

nama_saya = “Abubaba”
print nama_saya
umur=15
print umur
print umur +1

Output

Abubaba
15
16
