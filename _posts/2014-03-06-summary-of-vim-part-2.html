Rumus Umum :

Esc + Nomer + Perintah + Etc

Insert And Repeat Text

(Esc + 2 + i ) Ketikan Teks, Misal adalah ‘na’, lalu tekan Esc.

Output : nana.

Find Character on 1 Line

(Esc + f (next character), F (back character) + k) Mencari karakter K

Find Character { ( [ on 1 Line

(Esc + shift + %) Pindah pada karakter {[( di satu baris.

Move Cursor on Start Line and End Line

( Esc + 0) Pindah pada awal baris di satu baris.

( Esc + $ ) Pindah ke akhir baris di satu baris.

Move Line

(Esc + gg) Ke Line pertama

(Esc + G) Ke Line akhir

(Esc + 2 + gg) Kursor Pindah ke baris ke 2.
