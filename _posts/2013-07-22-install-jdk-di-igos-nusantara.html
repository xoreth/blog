http://download.oracle.com/otn-pub/java/jdk/7u25-b15/jdk-7u25-linux-x64.tar.gz >> 64bit

http://download.oracle.com/otn-pub/java/jdk/7u25-b15/jdk-7u25-linux-i586.tar.gz  >> 32bit

ekstrak dan simpan di /opt dan rubah nama jdk1.7.0_25 di /opt

sudo tar xvf jdk-7u25-linux-x64.tar.gz -C /opt
cd /opt

sudo mv jdk1.7.0_25 jdk

ubah kepemilikan

sudo chown namaUser:groupUser -R jdk/

Kembali ke /home/user

cd ~/

lalu daftarkan path direktori jdk pada konfigurasi shell bash/zsh di ~/.zshrc atau di ~/.bashrc

export PATH=$PATH:/opt/jdk:.

Jika sudah menulis simpan keluar lalu restart atau ketikan

resource  ~/.zshrc

untuk reload atau membaca ulang path.

ketikan :

java -version
