Struktur standar :

if (kondisi):

statement

elif (kondisi):

statment

….

else:

statment

Contoh :

isi=“ganteng”
if (isi==“ganteng”):
print “Dia tampan”
else:
print “Dia tidak tampan”

Output : Dia tampan
