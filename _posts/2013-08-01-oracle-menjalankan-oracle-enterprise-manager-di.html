Menjalankan Oracle Enterprise Manager Database di #igosnusantara

disable    host :
xhost +

masuk ke user oracle :
su oracle

jalankan listener:
lsnrctl    start

hidupkan database :
dbstart

Jika service ini belum berjalan    lihat,
[1]http://blog.wajatmaka.com/post/57059655515/membuat-listener-oracle-11g-di-igos-nusantara
[2]http://blog.wajatmaka.com/post/57094536163/menjalankan-service-database-di-igos-nusantara

setelah    berhasil, selanjutnya :
emctl start dbconsole

jika output adalah :
Oracle Enterprise Manager 11g Database Control Release 11.2.0.1.0
Copyright © 1996, 2009 Oracle Corporation.  All rights reserved.
https://igosnusantara:1158/em/console/aboutApplication
Oracle Enterprise Manager 11g is running.
——————————————————————
Logs are generated in directory /opt/oracle/product/11.2.0/db_1/igosnusantara_basedata/sysman/log

maka  berhasil.

lalu akses via web browser :
https://igosnusantara:1158/em/console/aboutApplication
atau
https://127.0.0.1:1158/em/console/aboutApplication
