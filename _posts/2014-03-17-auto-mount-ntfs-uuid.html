Mengkaitkan hardisk yang memilki file system NTFS pada saat booting dengan universally unique identifier (UUID).

sudo blkid

output : UUID=”xxxxxxxxxxxxxxxxx”

Contoh, jika akan mengkaikan di /MainGarage, tujuannya agar tidak jauh masuk ke dalam direktori jika membutuhkan sesuatu pada partisi yang akan dikaitkan ini.

sudo mkdir /MainGarage

tambahkan di /etc/fstab

UUID=xxxxxxxxxxxxxxxxx /MainGarage ntfs-3g defaults 0 0

Restart
